## Selection Notes

### Best All-Around Choices (Recommended Starting Set)

**Small Tier:**
- Java: `JSON-java`
- C#: `Clean Architecture Manga`
- C++: `CLI11`
- Python: `Click`

**Medium Tier:**
- Java: `Spring PetClinic`
- C#: `eShopOnWeb`
- C++: `Crow`
- Python: `FastAPI Full Stack`

**Large Tier:**
- Java: `Spring Boot Admin`
- C#: `OrchardCore`
- C++: `fish-shell`
- Python: `Django Oscar`

### Criteria Used
✅ Active maintenance (recent commits)
✅ Good existing test coverage
✅ Standard build tooling for language
✅ Realistic code quality (mix of good and improvable areas)
✅ Public, permissive licensing
✅ Documented (but not over-documented)
✅ Community-recognized projects

### Notes on Very Large Projects
Projects marked with "WARNING: Very large" may:
- Take longer to clone and analyze
- Require more disk space (1GB+)
- Have complex build requirements
- Be better suited for advanced benchmarking only
