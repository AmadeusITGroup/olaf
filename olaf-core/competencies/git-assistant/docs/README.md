# Git Assistant Competency Package

## Overview

Git workflow management and branch operations competency package focusing on feature extraction, branch management, safe merging with tagging, and pull request workflows

## Entry Points

### Create Feature For Pr

**Command**: `create feature`

**Protocol**: Propose-Confirm-Act

**Use Case**: Extract features from integration branch to create a feature branch and PR to main

- [Description](./create-feature-for-pr/description.md)
- [Tutorial](./create-feature-for-pr/tutorial.md)

### Propose Commit Thread

**Command**: `propose commit`

**Protocol**: Propose-Confirm-Act

**Use Case**: Analyze git changes, cluster into logical commits, and execute commit thread with interactive modification and GitHub issue integration

- [Description](./propose-commit-thread/description.md)
- [Tutorial](./propose-commit-thread/tutorial.md)

### Merge Branch With Safety

**Command**: `merge branch`

**Protocol**: Propose-Confirm-Act

**Use Case**: Perform safe branch merge with dry-run validation, automatic tagging before/after, and rollback capability

- [Description](./merge-branch-with-safety/description.md)
- [Tutorial](./merge-branch-with-safety/tutorial.md)
