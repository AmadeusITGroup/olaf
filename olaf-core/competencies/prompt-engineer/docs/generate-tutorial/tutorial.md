# Tutorial: Generate Tutorial

## Overview
Create comprehensive step-by-step tutorials from conversations or workflow files automatically.

## Prerequisites
- Source content (conversation or file)
- OLAF framework installed
- Understanding of workflow being documented

## Estimated Time
15-20 minutes

## Steps

### Step 1: Invoke Generate Tutorial
```
olaf generate tutorial
```

### Step 2: Select Source Type
Choose "current_conversation" or "file" as source.

### Step 3: Provide Workflow Name
Specify a descriptive name for the workflow being documented.

### Step 4: Specify Source File (if applicable)
If source_type is "file", provide the file path.

### Step 5: Select Target Language (Optional)
Choose language for tutorial content (default: English).

### Step 6: Review Generated Tutorial
Examine the tutorial for completeness and accuracy.

### Step 7: Save and Test Tutorial
Save the tutorial and test it with a new user.
