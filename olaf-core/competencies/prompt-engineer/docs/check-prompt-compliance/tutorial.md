# Tutorial: Check Prompt Compliance

## Overview
Validate prompts against OLAF standards and receive detailed compliance analysis with improvement recommendations.

## Prerequisites
- Prompt file to analyze
- OLAF framework installed
- Access to prompt template and principles

## Estimated Time
10-15 minutes

## Steps

### Step 1: Invoke Check Prompt Compliance
```
olaf check prompt compliance
```

### Step 2: Provide Target Prompt Path
Specify the path to the prompt you want to analyze.

### Step 3: Select Output Format (Optional)
Choose "summary", "detailed", or "checklist" format.

### Step 4: Review Compliance Report
Examine the compliance score, findings, and recommendations.

### Step 5: Address Identified Issues
Implement high-priority improvements first.

### Step 6: Re-validate After Changes
Run compliance check again to verify improvements.

## Verification Checklist
- [ ] Compliance score calculated
- [ ] Template sections evaluated
- [ ] Principles adherence assessed
- [ ] Improvement recommendations provided
- [ ] Issues prioritized

## Troubleshooting
- **Prompt file not accessible**: Check file path and permissions
- **Competency auto-detection failed**: Manually specify competency

## Next Steps
- Implement recommended improvements
- Re-run compliance check to verify fixes
- Document any intentional deviations from standards
