# Tutorial: Create Competency Package

## Overview
Build complete, self-contained OLAF competency packages with proper structure, manifest, and documentation.

## Prerequisites
- OLAF framework installed
- Write access to competencies directory
- Understanding of package purpose and target users

## Estimated Time
40-60 minutes

## Steps

### Step 1: Invoke Create Competency Package
```
olaf create competency package
```

### Step 2: Provide Package Name
Specify a kebab-case name for the package.

### Step 3: Describe Package Purpose
Provide a brief description of the package's capabilities.

### Step 4: Select Category
Choose appropriate category (e.g., "development-tools", "documentation").

### Step 5: Define Target Users
Specify primary and secondary user personas.

### Step 6: List Initial Prompts (Optional)
Provide names of prompts to include initially.

### Step 7: Review Package Structure
Examine the created directory structure and manifest.

### Step 8: Add Prompts and Templates
Populate the package with actual prompts and templates.

## Verification Checklist
- [ ] Package directory created
- [ ] Manifest properly formatted
- [ ] Folder structure complete
- [ ] README created
- [ ] Documentation structure in place

## Troubleshooting
- **Package already exists**: Choose different name
- **Invalid category**: Select from standard categories
