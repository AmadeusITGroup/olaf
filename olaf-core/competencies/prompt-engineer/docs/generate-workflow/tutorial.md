# Tutorial: Generate Workflow

## Overview
Create structured workflow documents from templates for sequential, iterative, decision-based, or orchestrator patterns.

## Prerequisites
- Understanding of process to formalize
- OLAF framework installed
- Knowledge of workflow participants

## Estimated Time
25-35 minutes

## Steps

### Step 1: Invoke Generate Workflow
```
olaf generate workflow
```

### Step 2: Select Workflow Type
Choose "sequential", "iterative", "decision", or "orchestrator".

### Step 3: Provide Workflow Name
Specify a descriptive name for the workflow.

### Step 4: Describe Workflow Purpose
Explain what the workflow accomplishes.

### Step 5: Provide Type-Specific Parameters
- Sequential: Number of phases
- Iterative: Iteration criteria
- Decision: Decision points
- Orchestrator: Sub-workflows

### Step 6: Review Generated Workflow
Examine phases, decision points, and success criteria.

### Step 7: Customize and Save
Make any necessary adjustments and save the workflow.

## Verification Checklist
- [ ] Workflow type appropriate
- [ ] All phases defined
- [ ] Success criteria clear
- [ ] Error handling included
- [ ] Decision logic explicit (if applicable)

## Troubleshooting
- **Wrong workflow type**: Restart with correct type
- **Missing phases**: Add manually to generated workflow
